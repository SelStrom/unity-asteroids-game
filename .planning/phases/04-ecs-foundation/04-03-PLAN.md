From Assets/Scripts/Model/Systems/GunSystem.cs:
```csharp
protected override void UpdateNode(GunComponent node, float deltaTime)
{
    if (node.CurrentShoots < node.MaxShoots)
    {
        node.ReloadRemaining -= deltaTime;
        if (node.ReloadRemaining <= 0)
        {
            node.ReloadRemaining = node.ReloadDurationSec;
            node.CurrentShoots = node.MaxShoots;
        }
    }

    if (node.Shooting && node.CurrentShoots > 0)
    {
        node.CurrentShoots--;
        node.OnShooting?.Invoke(node);
    }

    node.Shooting = false;
}
```

From Assets/Scripts/Model/Systems/LaserSystem.cs:
```csharp
protected override void UpdateNode(LaserComponent node, float deltaTime)
{
    if (node.CurrentShoots.Value < node.MaxShoots)
    {
        node.ReloadRemaining.Value -= deltaTime;
        if (node.ReloadRemaining.Value <= 0)
        {
            node.ReloadRemaining.Value = node.UpdateDurationSec;
            node.CurrentShoots.Value += 1;
        }
    }

    if (node.Shooting && node.CurrentShoots.Value > 0)
    {
        node.CurrentShoots.Value -= 1;
        node.OnShooting?.Invoke(node);
    }

    node.Shooting = false;
}
```

Key difference: GunSystem перезаряжает ВСЕ патроны сразу (CurrentShoots = MaxShoots), LaserSystem перезаряжает ПО ОДНОМУ (CurrentShoots += 1).
