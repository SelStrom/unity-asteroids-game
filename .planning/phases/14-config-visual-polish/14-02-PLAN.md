Через MCP выполнить следующие операции (per D-07, D-12, D-13):

1. Открыть префаб для редактирования:
```bash
unity-mcp-cli run-tool assets-prefab-open --input '{"gameObjectRef": {"instanceID": 0, "assetPath": "Assets/Media/prefabs/rocket.prefab"}}'
```

2. Получить список компонентов корневого объекта (для получения instanceID):
```bash
unity-mcp-cli run-tool gameobject-component-list-all --input '{"gameObjectRef": {"instanceID": 0, "assetPath": "Assets/Media/prefabs/rocket.prefab"}}'
```

3. Создать дочерний пустой GameObject с именем "Trail":
```bash
unity-mcp-cli run-tool gameobject-create --input '{"name": "Trail", "parentGameObjectRef": {"instanceID": 0, "assetPath": "Assets/Media/prefabs/rocket.prefab"}}'
```

4. Добавить компонент ParticleSystem на Trail GameObject:
```bash
unity-mcp-cli run-tool gameobject-component-add --input '{"gameObjectRef": {"name": "Trail"}, "componentType": "UnityEngine.ParticleSystem"}'
```

5. Настроить ParticleSystem через gameobject-component-modify (per D-13). Ключевые параметры:
   - `playOnAwake` = false (управляем вручную из RocketVisual)
   - `main.simulationSpace` = World (чтобы след оставался на месте)
   - `main.startLifetime` = 0.4
   - `main.startSize` = 0.08
   - `main.startColor` = белый (255, 255, 255, 200)
   - `emission.rateOverTime` = 40
   - `main.startSpeed` = 0
   - `main.maxParticles` = 50
   - `shape` включить, тип Cone, angle = 0
   - `sizeOverLifetime` включить, убывающая кривая
   - `colorOverLifetime` включить, fade to transparent

   Использовать gameobject-component-modify для каждой группы настроек. Если MCP не поддерживает все вложенные свойства ParticleSystem -- настроить максимум доступных через MCP, остальные отметить для ручной проверки.

6. Привязать _trailEffect на RocketVisual к дочернему Trail/ParticleSystem:
```bash
unity-mcp-cli run-tool gameobject-component-modify --input '{
  "gameObjectRef": {"instanceID": 0, "assetPath": "Assets/Media/prefabs/rocket.prefab"},
  "componentRef": {"typeName": "SelStrom.Asteroids.RocketVisual"},
  "componentDiff": {"typeName": "SelStrom.Asteroids.RocketVisual", "fields": [{"typeName": "UnityEngine.ParticleSystem", "name": "_trailEffect", "value": {"instanceID": TRAIL_PS_INSTANCE_ID}}]}
}'
```
(Использовать instanceID ParticleSystem, полученный на шаге 4)

7. Сохранить и закрыть префаб:
```bash
unity-mcp-cli run-tool assets-prefab-save --input '{"gameObjectRef": {"instanceID": 0, "assetPath": "Assets/Media/prefabs/rocket.prefab"}}'
unity-mcp-cli run-tool assets-prefab-close --input '{"gameObjectRef": {"instanceID": 0, "assetPath": "Assets/Media/prefabs/rocket.prefab"}}'
```

ВАЖНО: instanceID значения динамические -- использовать значения из ответов MCP на каждом шаге. Порядок операций строго последовательный.
