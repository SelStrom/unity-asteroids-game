**Шаг 1: Установка URP пакета (per D-01, D-02)**

Добавить в `Packages/manifest.json` зависимость:
```json
"com.unity.render-pipelines.universal": "17.0.5"
```
Пакет из Unity Registry, точную версию для Unity 6.3 определить из packages-lock.json после резолва. Если 17.0.5 не резолвится, использовать версию доступную в Unity 6.3 Registry (17.5.x).

**Шаг 2: Создание URP ассетов (per D-02)**

Создать директорию `Assets/Settings/`.

Создать `Assets/Settings/URP-2D-Renderer.asset` -- 2D Renderer Data Asset. Это ScriptableObject типа `UnityEngine.Rendering.Universal.Renderer2DData`. Создать YAML-файл с корректной структурой Unity ассета.

Создать `Assets/Settings/URP-2D-Asset.asset` -- URP Pipeline Asset. Это ScriptableObject типа `UnityEngine.Rendering.Universal.UniversalRenderPipelineAsset`, ссылающийся на 2D Renderer Data.

ВАЖНО: Рекомендуемый подход -- использовать Unity Editor (через MCP или вручную) для создания ассетов через меню Create > Rendering > URP Asset (with 2D Renderer). Если Unity Editor недоступен, создать минимальные YAML-заглушки и пометить для ручного пересоздания.

**Шаг 3: Назначение URP в Project Settings (per D-02)**

В `ProjectSettings/GraphicsSettings.asset`:
- Заменить `m_CustomRenderPipeline: {fileID: 0}` на ссылку с GUID созданного URP-2D-Asset.asset

В `ProjectSettings/QualitySettings.asset`:
- Найти ВСЕ 6 Quality Level (Very Low, Low, Medium, High, Very High, Ultra)
- В КАЖДОМ заменить `customRenderPipeline: {fileID: 0}` на ссылку с GUID URP-2D-Asset.asset
- Текущий активный уровень: Ultra (m_CurrentQuality: 5)

**Шаг 4: Создание URP материалов (per D-06, D-08)**

Создать директорию `Assets/Media/materials/`.

Создать `Assets/Media/materials/Laser-URP.mat`:
- Шейдер: `Universal Render Pipeline/Unlit`
- Цвет: белый (1,1,1,1) -- как в оригинале (gradient key0/key1 = white)
- Surface Type: Opaque

Создать `Assets/Media/materials/Particle-URP.mat`:
- Шейдер: `Universal Render Pipeline/Particles/Unlit`
- Surface Type: Transparent
- Blending Mode: Additive (для эффекта взрыва)
- Base Color: белый (1,1,1,1)

**Шаг 5: Назначение материалов в prefabs (per D-06, D-08)**

В `Assets/Media/effects/lazer.prefab`:
- Заменить ссылку материала LineRenderer с `{fileID: 10306, guid: 0000000000000000f000000000000000}` (Default-Line) на ссылку на `Assets/Media/materials/Laser-URP.mat`

В `Assets/Media/effects/vfx_blow.prefab`:

- Убедиться, что `stopAction: 3` (Callback) сохранен (per D-07)

**Шаг 6: Настройка Post-Processing (per D-10, D-11)**

Создать `Assets/Settings/PostProcessing-Profile.asset` -- Volume Profile ScriptableObject:
- Bloom override: threshold=0.9, intensity=0.5, scatter=0.7
- Vignette override: color=black(0,0,0), center=(0.5,0.5), intensity=0.35, smoothness=0.4

В `Assets/Scenes/Main.unity`:
- Добавить новый GameObject "PostProcessing Volume" с компонентом Volume (isGlobal=true), назначить PostProcessing-Profile.asset
- На Main Camera включить Post Processing: найти UniversalAdditionalCameraData и установить renderPostProcessing=true (per D-13 -- компонент добавится автоматически при назначении URP Pipeline, но Post Processing checkbox нужно включить явно)
- Сохранить параметры камеры: orthographicSize=22.5 (per D-12)

**Шаг 7: Render Pipeline Converter для спрайтов (per D-03, D-04)**

Спрайты с Sprites-Default (fileID: 10754) в gameplay prefabs (ship, asteroids, bullets, ufo) должны автоматически работать с URP -- 2D Renderer подхватывает Sprite-Unlit-Default без явной конвертации материалов. Если при запуске спрайты розовые, запустить Window > Rendering > Render Pipeline Converter > Built-In 2D to URP 2D.

ВАЖНО: НЕ использовать 2D Lighting и Sprite-Lit-Default -- это отложено на VIS-01 (per Deferred Ideas). Использовать Sprite-Unlit-Default для 1:1 соответствия.

using NUnit.Framework;
using UnityEngine;
using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

namespace SelStrom.Asteroids.Tests.EditMode
{
    [TestFixture]
    public class UrpSetupTests
    {
        [Test]
        public void UrpPipelineAssetExists()
        {
            // Assets/Settings/URP-2D-Asset.asset должен существовать
            var asset = UnityEditor.AssetDatabase.LoadAssetAtPath<UniversalRenderPipelineAsset>(
                "Assets/Settings/URP-2D-Asset.asset");
            Assert.IsNotNull(asset, "URP Pipeline Asset должен существовать в Assets/Settings/URP-2D-Asset.asset");
        }

        [Test]
        public void RendererDataAssetExists()
        {
            // Assets/Settings/URP-2D-Renderer.asset должен существовать
            var asset = UnityEditor.AssetDatabase.LoadAssetAtPath<ScriptableRendererData>(
                "Assets/Settings/URP-2D-Renderer.asset");
            Assert.IsNotNull(asset, "2D Renderer Data должен существовать в Assets/Settings/URP-2D-Renderer.asset");
        }

        [Test]
        public void GraphicsSettingsUsesUrp()
        {
            var pipeline = GraphicsSettings.currentRenderPipeline;
            Assert.IsNotNull(pipeline, "GraphicsSettings.currentRenderPipeline должен быть назначен (не null)");
            Assert.IsInstanceOf<UniversalRenderPipelineAsset>(pipeline,
                "currentRenderPipeline должен быть UniversalRenderPipelineAsset");
        }

        [Test]
        public void QualitySettingsAllLevelsHaveUrp()
        {
            var qualityLevelCount = QualitySettings.names.Length;
            var currentLevel = QualitySettings.GetQualityLevel();

            for (int i = 0; i < qualityLevelCount; i++)
            {
                QualitySettings.SetQualityLevel(i, false);
                var pipeline = QualitySettings.renderPipeline;
                Assert.IsNotNull(pipeline,
                    $"Quality Level {i} ({QualitySettings.names[i]}) должен иметь назначенный URP Asset");
            }

            QualitySettings.SetQualityLevel(currentLevel, false);
        }
    }
}
