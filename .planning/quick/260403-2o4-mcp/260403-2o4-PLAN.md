<objective>
Вернуть пакет com.ivanmurzak.unity.mcp в manifest.json и настроить MCP-сервер
в Claude Code settings для прямого взаимодействия с Unity Editor (запуск тестов,
проверка компиляции, чтение Console).
</objective>

<tasks>
<task type="auto">
  <name>Task 1: Добавить Unity-MCP пакет и настроить MCP сервер</name>
  <action>
  1. Добавить com.ivanmurzak.unity.mcp в Packages/manifest.json
  2. Создать .mcp.json с HTTP transport на порт 50972
  3. Добавить mcp__unity-mcp__* permission в .claude/settings.local.json
  </action>
</task>
</tasks>
